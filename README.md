# Josh-a-nator

A dumb little Angular page that puts Josh quotes in a speech bubble.

**Live:** https://mfontecchio.github.io/josh-a-nator/

## Development

```bash
npm start
```

Open http://localhost:4200/

## Build

```bash
npm run build
```

Production builds use `baseHref` `/josh-a-nator/` for GitHub Pages.

## Deploy

Pushes to `main` deploy via GitHub Actions to GitHub Pages.
